# 欢迎向导浅色主题设计

## 1. 问题描述

当前欢迎向导使用深色主题（左侧 `#1E1E1E`、右侧 `#252525`），用户希望改成 Windows 风格的浅色主题。

## 2. 目标

将 `WelcomeWindow` 的整体配色改为 Windows 11 设置向导风格的浅色主题，保持现有布局和交互不变。

## 3. 配色方案

参考 Windows 11 设置应用：

| 元素 | 当前颜色 | 目标颜色 |
|------|----------|----------|
| 窗口背景 | `#252525` | `#F3F3F3` |
| 左侧导航栏背景 | `#1E1E1E` | `#FFFFFF` |
| 主标题文字 | `White` | `#1F1F1F` |
| 副标题/说明文字 | `#CCFFFFFF` / `#AAFFFFFF` | `#5F5F5F` |
| 强调色/当前步骤 | `#4A90E2` | `#0078D4` |
| 未到达步骤指示器 | `#55FFFFFF` | `#C4C4C4` |
| 未到达步骤文字 | `#AAFFFFFF` | `#8A8A8A` |
| 底部按钮区背景 | `#2D2D2D` | `#FFFFFF` |
| 底部按钮区边框 | 无 | `#E5E5E5` 1px 上边框 |
| 主按钮（下一步/完成） | 默认 | 背景 `#0078D4`，文字 `White` |
| 次按钮（上一步） | 默认 | 背景 `#E5E5E5`，文字 `#1F1F1F` |
| Logo 背景 | `#4A90E2` | `#0078D4` |
| Logo 文字 | `White` | `White` |
| 应用名称 | `White` | `#1F1F1F` |
| "设置向导" 文字 | `White` 60% 透明 | `#8A8A8A` |
| 复选框文字 | `White` | `#1F1F1F` |

## 4. 实现范围

仅修改 `FullCompo.App/Views/WelcomeWindow.axaml` 中的颜色和少量样式，不动 C# 逻辑。

## 5. 验收标准

- [ ] 欢迎向导整体为浅色风格
- [ ] 左侧导航栏白色背景，深色文字
- [ ] 右侧内容区浅灰背景
- [ ] 步骤指示器当前步骤为 Windows 蓝色
- [ ] 底部按钮区有浅色上边框
- [ ] "下一步"/"完成"按钮为蓝色
